LEADER KEY: Space (" ")

--------------------------------------------------------------------------------
NOTE ON ADDING PLUGINS
--------------------------------------------------------------------------------
When you add a new plugin:
  1. Add the plugin spec under `lua/plugins/`.
  2. Register it in `init.lua` inside `require("lazy").setup(...)`.
  3. Document any new keymaps here in `keybinds.txt`.
  4. Make sure the README stays in sync with any setup or usage changes.

Recommended plugin checklist:
  - Keep the spec minimal and consistent with the other plugin files.
  - Prefer lazy-loading unless the plugin must load at startup.
  - Add or update key descriptions so mappings are easy to discover.
  - Verify the plugin's keys do not conflict with existing bindings.

GENERAL (init.lua)
--------------------------------------------------------------------------------
Splits
  <leader>v          Open vertical split
  <leader>b          Open horizontal split
  <leader>q          Close current split

Window Navigation
  <C-h>             Move to left window
  <C-j>             Move to bottom window
  <C-k>             Move to top window
  <C-l>             Move to right window

Window Resize
  <A-h>             Shrink width (vertical resize -2)
  <A-l>             Grow width (vertical resize +2)
  <A-k>             Grow height (resize +2)
  <A-j>             Shrink height (resize -2)

Tab Management
  <leader>tn         New tab
  <leader>tc         Close tab
  <C-Tab>            Next tab
  <C-S-Tab>          Previous tab

Jump List
  <C-o>             Jump back
  <leader>j         Jump forward
  <leader>jl        View jump list

Gitsigns
  ]c                Next hunk
  [c                Previous hunk
  <leader>hs        Stage hunk
  <leader>hr        Reset hunk
  <leader>hp        Preview hunk
  <leader>hb        Toggle blame line
  <leader>hd        Diff this file
